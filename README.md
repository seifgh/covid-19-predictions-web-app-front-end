### Prerequisites

- npm package manager 

```
$ npm --version
6.13.4

```
- NodeJs JavaScript runtime environment  
```
$ node --version
v12.16.1


```
---
### Installing

- Install dependencies
```terminal
$ npm install
```
---
### Running
```terminal
$ npm start // for running react app (front-end)
$ python manage.py runserver // for running django (back-end)
```
